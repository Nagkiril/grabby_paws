# Grabby Paws

A game I'm making solo while pushing some code publicly.

The premise is super simple:
- You walk with a Character you control in Top-Down\3rd Person camera.
- Your objective is to collect Treasure while avoiding Hazards.

Please switch to a version branch for actual code, changelogs and state of the project.

## Technical Considerations

Like previously [Slime Bounce](https://github.com/Nagkiril/SlimeBounce_public), I mainly develop in another private repo, and here I'll be making occasional version updates.

I'd like to keep this project functional so that anyone can get the code and play it, but:
1. I have my own coding preferences, which may have some code look weirdly formatted. When working in teams I adapt to their preferences, but I have my own elaborate reasoning for X over Y when there's no one else working with me.
3. Some things may look weird, as I'll be using some packages I'm not allowed to redistribute (i.e. Feel).
4. When I'll be getting closer to a satisfactory result, I may choose to publish the game and stop updating this repository (but then I'll post a link to the game instead).
